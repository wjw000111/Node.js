nvm
    -命令：
        nvm list -显示已安装的node版本
        nvm install 版本 -安装执行版本的Node
            nvm install latest 下载最新版本
            nvm install lts 下载稳定版
        配置nvm的镜像服务器
            nvm node_mirror https://npmmirror.com/mirrors/node/
        nvm use 版本 -指定要使用的node的版本
